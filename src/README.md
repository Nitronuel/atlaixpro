# Application Source

This folder contains the Atlaix React application, shared TypeScript types, intelligence services, page-level routes, reusable components, tests, and utility helpers.

Key areas:

- `pages/` contains route-level product surfaces.
- `services/` contains provider clients, scoring engines, scanner logic, and data adapters.
- `components/` contains reusable UI and feature components.
- `types/` centralizes shared platform types.
- `utils/` contains formatting, chain, and wallet helpers.
